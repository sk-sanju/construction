# Art Home Construction Website

A modern, premium, fully responsive website designed for **Art Home Construction**, a top-tier construction builder based in Pazhayannur, Thrissur, Kerala.

## Technical Architecture

- **Front-end Structure**: HTML5 semantic coding with SEO meta indexing.
- **Responsive Layout**: Bootstrap v5.3 + Custom CSS Flexbox/Grid offsets.
- **Typography & Icons**: Poppins (Google Fonts) and Bootstrap Icons.
- **Interactions**: Vanilla JS (Loader state, scroll indicator, stats counter animation using Intersection Observer API, portfolio tabs filter, floating CTAs, form validations).
